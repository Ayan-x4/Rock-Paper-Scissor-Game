# ✊ Rock Paper Scissors Game

A simple and fun **Rock Paper Scissors** game built using basic web technologies. This project allows users to play against the computer with instant Result.

## 🚀 Features

* Play Rock, Paper, or Scissors with a single click
* Random computer choice generation
* Instant result display (Win / Lose / Draw)

🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🎮 How to Play

1. Choose Rock, Paper, or Scissors
2. The computer will automatically make a random choice
3. The result will be displayed instantly:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock

## 📌 Purpose

This project is great for beginners to understand:

* DOM manipulation
* Event handling
* Basic game logic in JavaScript
